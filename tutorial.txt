COMO USAR A API 2.0


Dicas Úteis para o primeiro uso:

1 - Usar postman ou outra ide de sua preferencia e setar o body como  x-www-form-urlenconded
2 - METODO POST - http://evorgaapi-com.umbler.net/api/cadastrarUsuario  (rotas estão dentro da pasta routes e por enquanto tudo dentro de api)
2.1 - Parâmetro para o POST 
      username
      password
      email
      photoUpload - não obrigatório
      id_Perfil - não obrigatório 
3 - METODO POST - http://evorgaapi-com.umbler.net/api/logar
3.1 - Parâmetro para o POST 
      username
      password
3.2 (importante) Ele vai retornar um token, copiar este token
3.3  no header da requisição colocar uma key com o nome Authorization e no valor o token copiado

4- Generico - MetodoPOST http://evorgaapi-com.umbler.net/api/cadastrarNomeDaModel e passar parametros (no arquivo api , dentro da pasta routes você pode verificar)
5- Generico - Metodo GET http://evorgaapi-com.umbler.net/api/getAllNomeDaModelNoPlural
6 -Generico - Metodo POST http://evorgaapi-com.umbler.net/api/getNomeDaModel e passar id como parametro
